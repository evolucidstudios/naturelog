Backend project spec

This or That

This concept needs PHP + MySQL for persistent voting and admin-created matchups, so this shelf entry is a project sheet rather than a runnable page on this host.

Suggested structure

  • index.php active matchup and voting
  • results.php previous matchups and totals
  • admin.php create matchups
  • vote.php AJAX vote endpoint
  • schema.sql matchups, options, votes
